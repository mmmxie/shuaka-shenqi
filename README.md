# 刷卡神器

台灣信用卡回饋比較工具，專為在英國的台灣人設計。

## 功能

- 比較多張信用卡在不同情境（英國、台灣、日本、韓國、歐洲等）的回饋率
- 自動計算海外手續費後的實際回饋
- 支援月消費上限追蹤（含自動帳期重置）
- 所有資料存於本地 localStorage，無需伺服器

## 使用方式

直接用瀏覽器開啟 `刷卡神器.html`，無需安裝任何套件。

## 自動更新

`daily_card_update.sh` 每天 00:00 自動檢查信用卡回饋資料是否有更新，透過 macOS crontab 執行。

## 技術架構

單一 HTML 檔，純 vanilla JS + CSS，無外部 framework 依賴。
詳細架構說明見 `docs/` 資料夾。
